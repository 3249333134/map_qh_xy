mysql:
sudo docker run -d \
  --name mysql-server \
  --restart=always \
  -p 3306:3306 \
  -v mysql-data:/var/lib/mysql \
  -e MYSQL_ROOT_PASSWORD=xumin999 \
  -e MYSQL_ROOT_HOST=% \
  mysql:8.0

* -d：后台运行容器
* --name mysql-server：给容器命名为 mysql-server（方便管理）
* --restart=always：Docker 重启时自动启动该容器
* -p 3306:3306：端口映射，宿主机 3306 端口映射到容器内 3306 端口（外部可通过宿主机 IP:3306 访问）
* -v mysql-data:/var/lib/mysql：数据卷挂载，将容器内 MySQL 数据目录挂载到宿主机数据卷，实现数据持久化
* -e MYSQL_ROOT_PASSWORD=xumin999： MySQL root 用户密码
* -e MYSQL_ROOT_HOST=%：允许 root 用户从任意主机远程访问


Redis:
sudo docker run -d \
  --name redis-server \
  --restart=always \
  -p 6379:6379 \
  -v /usr/local/redis/conf/redis.conf:/etc/redis/redis.conf \
  -v /usr/local/redis/data:/data \
  redis:7.0 redis-server /etc/redis/redis.conf

* --name redis-server：容器命名，方便管理
* --restart=always：Docker 重启时自动启动 Redis
* -p 6379:6379：端口映射，宿主机 6379→容器 6379
* -v /usr/local/redis/conf/redis.conf:/etc/redis/redis.conf：挂载自定义配置文件（覆盖容器默认配置）
* -v /usr/local/redis/data:/data：挂载数据目录（持久化 Redis 数据，避免容器删除丢失）
* redis-server /etc/redis/redis.conf：指定容器启动时加载自定义配置文件
