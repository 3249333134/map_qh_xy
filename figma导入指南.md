# 完整地图应用 Figma 原型导入指南

## 📁 文件说明

已生成 `complete-figma-prototype.html` 文件，这是一个包含**所有页面**的完整交互式HTML原型，一比一还原了你的地图应用的全部界面。

## 如何导入Figma

### 方法一：使用Figma插件（推荐）

1. **安装插件**
   - 在Figma中搜索并安装 "HTML to Figma" 插件
   - 或者使用 "Figmotion" 等其他HTML导入插件

2. **导入步骤**
   ```
   - 打开Figma设计文件
   - 按快捷键 Ctrl/Cmd + / 打开插件搜索
   - 搜索 "HTML to Figma"
   - 选择插件后，点击"Run"
   - 将本地HTML文件拖入插件窗口，或复制HTML代码粘贴
   - 等待转换完成
   ```

### 方法二：通过浏览器截图导入

1. **在浏览器中打开**
   ```
   - 用Chrome或Safari打开 figma-prototype.html
   - 按F11进入全屏模式
   - 使用系统截图工具或浏览器截图插件
   ```

2. **导入Figma**
   ```
   - 在Figma中：File > Place Image
   - 选择截图文件
   - 使用Figma的切片工具重新组织布局
   ```
